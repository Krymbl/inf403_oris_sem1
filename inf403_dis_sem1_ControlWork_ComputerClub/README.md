Разработать веб-приложение упакованное в war.

1. Приложение работает с БД (добавление, чтение, обновление), минимум 1 таблица
2. Страницы:
   главная (путь - /),
   просмотр всех (/show) в виде таблицы
   просмотр выбранного (/showone?id=..) в виде заполненной формы с возможностью сохранить,
   добавление нового элемента (get запрос, путь /add),_показываем пустую форму
   добавление нового элемента (post запрос, путь /add),_отправляем на этот путь данные формы, потом отображаем весь список

Предметную область выбирайте самостоятельно, в таблице опишите не менее 5 атрибутов
